# Congreso Aeroespacial 2026

Sitio web estatico de trabajo para `Congreso Nacional ENMICE 2026` / `Desarrollo e Innovacion Aeroespacial`.

## Estructura

- `index.html`: pagina principal
- `404.html`: pagina de error para GitHub Pages
- `assets/scss/`: fuente de estilos
- `assets/css/main.css`: CSS compilado desde SCSS

## Notas

- La fuente de verdad para estilos es `assets/scss/`.
- Los enlaces de registro y envio de trabajos siguen apuntando a placeholders de Google Forms hasta tener la liga final.
- Esta carpeta esta preparada para usarse como repo independiente y para publicar con GitHub Pages.
